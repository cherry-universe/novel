5. 你裙子裂开啦

正午的太阳平等地向每一个人散发着她的热量。林橙仍在与那件“初中珍藏品”搏斗，紧勒的腰线和飘忽的裙摆迫使她维持着僵硬的步态。

“姐，你看那边！”林小雨突然指向前方路口闪烁的蓝红警灯和围观人群，“出啥事了？车祸？”

好奇心驱使两人挤进人群。眼前的景象让林橙倒吸一口凉气——路中央，一只体型如小汽车半截、形似巨大史莱姆的粘稠生物正缓慢蠕动。它半透明的躯体分泌着腐蚀性粘液，所过之处柏油路面滋滋作响，冒出刺鼻白烟，几颗幽绿的瘤状物点缀其上，闪烁着不祥的光。

“卧槽！鼻涕虫？！”林小雨惊呼着躲到林橙身后。

“是‘蚀泥怪’！”一个身穿“达利安异能管理局-外勤”制服的眼镜男正双手按地，额头青筋暴起。他面前不断隆起土墙试图阻挡，但粘液腐蚀性极强，土墙迅速被溶蚀得千疮百孔。

“F级污染型异化兽！物理攻击差，怕火和强能量！”另一个魁梧的火焰拳壮汉吼道，他双拳燃火试图进攻，却被甩来的粘液轻易浇灭火焰，狼狈后撤。

“妈的！‘灼热指虎’今天不给力！”壮汉脸色涨红。

“我的‘地脉涌动’快撑不住了！能量见底！支援呢？”眼镜男声音发颤。蚀泥怪的目标显然转向了路边一个报刊亭，蠕动速度似乎加快了些，一大滩粘液眼看就要糊上印着篮球封面的《知音》。

“完了完了，王大爷的亭子……” 壮汉绝望低吼。吃瓜群众的议论嗡嗡作响，充满了对低阶外勤的调侃和对怪物的惊奇。

“姐！亭子要没了！”林小雨紧张地抓紧林橙的胳膊。林橙心脏狂跳，顾不上自身能量的虚弱和对能力的生疏，目光死死锁住那逼近报刊亭的粘稠怪物。就在她集中精神的瞬间，视野猛地被强制刷新——

[./monster_book]

[version 0.15.3]（测试版，并且是GPL协议的开源软件，不接受投诉哦~）

[Target: 蚀泥怪 (ID: 3125768)]]

[Status: 龟速移动中 (速度: 0.5m/s)... 分泌超强酸性口水 (pH≈1)...]

[核心能量源: 低等异化晶核 (位置: 大概在中间那坨?)]

Solution suggestions:

1. apply_heat(extreme); // 来个太阳表面温度烧烤

2. apply_energy_shock(high_voltage); // 请它体验百万伏特杨教授疗法

3. freeze(target); // 速冻，做成史莱姆冰沙

4. delete(target.core); // 直接删号！(权限不足/风险极高)

5. set_speed(target, 0); // 给它按个暂停键 (耗蓝警告！需持续供能)

选项如弹幕飞过。烧烤？电疗？速冻？删号？都不可行！只剩最后一个看似“温和”的暂停键！时间紧迫，林橙意念急转，输入指令：

`set_speed(蚀泥怪, 0);`

指令生效的刹那，一股远比修改水晶球时狂暴的能量洪流瞬间从她体内被抽走。剧烈的眩晕和虚脱感让她眼前发黑，脚下踉跄，几乎站立不稳。

而场中，那只正“努力”扑向报刊亭的蚀泥怪，庞大粘稠的身躯猛地一顿！所有的蠕动、粘液分泌、甚至那滴即将滴落的腐蚀液珠，都如同被按下了绝对暂停的电影画面，彻底凝固在离亭子仅半米之处。

世界仿佛被按下了静音键，只有晚风掠过。

“啊？！”眼镜男感觉手下一空，茫然抬头，下巴几乎脱臼，“停…停了？”

“啥？！”火焰拳壮汉保持着冲锋姿势僵在原地，像座滑稽的雕塑，“它…它卡壳了？！”

围观人群爆发出更响的嗡嗡声，充满了难以置信的困惑：“怎么回事？”“谁干的？”“不像是那两位啊…”

只有林小雨，短暂的惊愕后猛地扭头看向身边脸色惨白、扶着额头摇摇欲坠的林橙。她那双卡姿兰大眼睛爆发出堪比探照灯的狂喜光芒，死死抓住林橙的手，用尽全力压抑尖叫，声音因激动而劈叉：“姐——！！！是你！绝对是你！你让它‘卡机’了？！代码暂停现实？！卧槽！你是我滴神！！！”

刺耳的警笛声由远及近，几辆喷涂管理局标志、蓝光更盛的车辆疾驰而至。车上跳下装备精良、气息沉稳的队员迅速接管现场。一个队长模样的中年男人走到凝固的蚀泥怪前，用仪器谨慎扫描，眉头紧锁：“能量场被强行凝固？没有元素残留，也不是精神禁锢…”他鹰隼般的目光锐利地扫过人群，最终精准地落在那位被妹妹搀扶着的、银发异瞳、穿着紧绷小裙子、一脸虚脱相的少女身上，眼神充满了深意。

眼镜男和壮汉也回过神来，一边协助收容那静止的怪物，一边心有余悸又困惑地不时瞟向林橙的方向。

危机解除，人群开始散去。林小雨几乎是半扛着“能量见底”的林橙，兴奋得语无伦次：“姐！你太神了！以后你就是我的专属外挂加防火墙！逛街再也不怕了…”

林橙有气无力地翻个白眼：“别…嚎了…我快散架了…这破裙子…”她感觉腰间的束缚已达极限，每一次呼吸都伴随着布料不堪重负的呻吟。

“叮咚！”林小雨的手机突然自动弹出管理局APP的通知。她下意识点开“附近异能波动”记录——刚才的位置，赫然标记着一个刺眼无比的峰值，旁边一行小字：

[检测到高能级未知异能波动，预估等级：S（待核实）]

林小雨倒吸一口冷气，猛地捂住手机屏幕，眼睛瞪得滚圆，僵硬地转向还在跟裙子做最后抗争的林橙，声音都变了调：“姐…姐！异管局APP…它…它觉得你是…S级？！”

“S级？！”林橙脑子嗡的一声，瞬间的震惊让她下意识吸了口气，本就紧绷的腰腹猛然收缩——

“嗤啦！”

一声清脆的布料撕裂声在喧闹渐散的街头格外清晰。

晚风带着胜利的凉意，欢快地灌入林橙腰间那道新鲜出炉的裂口，亲切地拂过她被迫穿上的、图案幼稚的草莓内裤边缘。

林小雨的尖叫瞬间刺破夜空：“啊——！！！姐！你裙子！裂！开！啦！！！”
