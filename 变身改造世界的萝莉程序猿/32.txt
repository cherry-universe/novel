32. Aha
2029年10月7日，夜。

林小雨房间的灯光已经调暗。屏幕上，维生舱内的淡蓝光芒是唯一的光源。林小雨已经趴在屏幕前的桌子上睡着了，脸上还带着未干的泪痕，嘴里无意识地呢喃着：“姐姐…别丢下我…哥哥…”

在她看不到的、林橙的维生舱内。

那具小小的身体依旧毫无动静。但在意识的最深处，那片被SS级能量爆发和自我献祭式回滚彻底“格式化”过的、近乎虚无的混沌之海深处。

绝对的寂静。绝对的黑暗。没有时间，没有空间，没有“我”的概念。

突然。

一点极其微弱的、幽蓝色的光点，如同坏掉的显示器上最后挣扎的一个像素，在无垠的虚无中闪烁了一下。

紧接着，一行扭曲的、跳动着癫狂色彩的、如同劣质街机游戏故障特效般的文字，凭空“打印”在这片意识虚空的背景之上：

// [阿哈] Process 'Sister_Revival.exe' completed with WARNINGS! ^o^
//   - Resource 'Ego_Boundary.sys' CRITICALLY LOW! (99.9% Corrupted) X_X
//   - Critical Error: Soul_Force.dll NOT FOUND! (Required for Operation) :'(
//   - Backup Volume 'Trauma_Core.img' mounted successfully! (Read-Only) \o/
//   - System State: KERNEL PANIC (Uninterruptible Sleep) Zzzz...
//   - Standby for next joke... Loading Humor_Module.dll (0%)... >:D

这行文字如同病毒般闪烁、扭曲，带着一种令人极度不适的、恶意满满的“欢快”感，持续了几秒钟，然后如同出现时一样突兀地，彻底消失了。

虚无的混沌之海，重新归于死寂。

核心控制室。
余胜军猛地从半睡半醒的状态中弹起来，布满血丝的眼睛死死盯住他的分析屏幕！
“郑队！快看！”他的声音因为激动和疲惫而劈叉。
郑峰一个箭步冲到他身后。
只见屏幕上，原本代表林橙基础生命体征（心跳、呼吸）的、两条几乎变成直线的微弱曲线，在刚才那一瞬间，极其诡异地、同步地向上跳动了一下！幅度虽然微小，但在长期近乎直线的背景下，显得异常突兀！
更关键的是，旁边一个被余胜军标记为“意识背景噪声模式异常”的次级监控窗口，捕捉到了一次极其短暂但高度有序的、类似“数据包突发”的尖峰脉冲！脉冲的波形，与余胜军之前注意到的、林小雨倾诉时出现的异常波形片段，有着某种难以言喻的相似性！

“波动！有反应了！”余胜军的声音带着难以置信的颤抖，“虽然…虽然看不懂是什么…但绝对不是随机的生理噪声！有东西…有东西在里面动了！”

郑峰的心跳漏了一拍。他立刻看向主监控屏上林橙维生舱的实时画面。

几乎就在同一时刻。
林小雨房间内。
睡梦中的林小雨似乎被某种强烈的情绪牵引，无意识地动了一下。一滴积蓄在眼角的泪珠，终于承受不住重量，悄然滑落，滴在她握着林橙冰凉手背的手指上。

温热的泪水，顺着林小雨的手指，轻轻浸润了林橙毫无知觉的、暴露在维生舱外的、同样冰凉的手背皮肤。

就在那滴泪水接触皮肤的瞬间。
维生舱内。
林橙那一直如同精致人偶般、静止地垂放在身侧、连接着数根生命管线的小手。
那纤细、苍白、毫无血色的食指指尖。
极其轻微地。
颤动了一下。
