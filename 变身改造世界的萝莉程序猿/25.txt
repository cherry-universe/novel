25. 前世

林小雨的身体在林橙怀中猛然一沉。温热的液体浸透了林橙的白裙前襟，浓重的铁锈味瞬间盖过小巷垃圾的酸腐气息。时间仿佛被强行挂起进程，林橙的视网膜上炸开一片猩红的错误提示：

// FATAL ERROR: Core_System_Threat_Detected
// Segmentation fault at address 0x00007ffd`b73e5f30
// Stack frame corrupted. Critical process terminated.

“姐……”林小雨的嘴唇翕动着，瞳孔里的光像断电的屏幕般迅速黯淡。混混头目手中的钢管还在滴血，他脸上混杂着错愕与暴戾，仿佛只是失手砸坏了一件不值钱的物品。

“小雨——！”林橙的嘶喊卡在喉咙里，变成一声破碎的气音。她想抬起手，指尖却像生锈的机械臂般僵硬。`sudo kill -9 [混混PID]`的命令在脑内疯狂编译，但枯竭的能量池如同被rm -rf格式化过的硬盘，连一个比特的响应都无法榨出。混混头目啐了一口，钢管再次扬起，阴影笼罩下来。

剧痛撕裂了林橙的意识屏障。不是物理的痛楚，而是灵魂深处某个加密分区被暴力破解——无数没有指针引用的记忆碎片，裹挟着冰冷的泥土与霉菌的气味，汹涌倒灌。

十三岁。地下室的霉斑在视野里摇晃。

十三岁的身体被粗糙的毯子摩擦着，劣质香水和汗液混合的酸臭几乎令人窒息。一只戴着金戒指的手捏住她的下巴，油腻的指肚陷进脸颊。“纯白的头发……稀罕货。”男人的声音像生锈的铰链，“好好‘养’，能卖大价钱。”

铁门在身后哐当锁死。黑暗成了唯一的被褥。她蜷缩在角落，指甲抠着水泥地缝，试图fork()一个子进程逃离此地。回应她的只有系统错误：
// Operation not permitted
// Permission denied (errno=13)

药物被混在馊饭里灌下。身体被强行停滞在十三岁的轮廓，内里的脏器却在化学毒剂的侵蚀下哀鸣。镜子里，那双被客人称赞为“宝石”的异色瞳日益黯淡，如同两枚即将耗尽电量的故障LED。

十四岁。铁链拴在暖气管道上。

“跑？打断你的腿！”看守的咆哮震得耳膜嗡嗡作响。上一个试图逃跑的女孩被拖回来时，双腿以诡异的角度扭曲着，像两条被拧断的数据线。绝望如同`while(1)`的死循环，将灵魂禁锢在这方寸之地。她开始对着墙壁无声地敲击莫尔斯电码，幻想自己是一段等待远程登录的守护进程。唯一的回应是老鼠啃噬垃圾的窸窣声，以及隔壁房间永不停止的、压抑的啜泣。

同样是十四岁。肮脏、逼仄的房间。

浓重的廉价香水、汗液和某种难以言喻的腥膻气味混合在一起，令人窒息。粗糙的麻绳深深勒进细嫩的手腕，火辣辣地疼。一个高大的黑影笼罩下来，带着令人作呕的酒气和狞笑。粗糙的大手像铁钳般抓住她纤细的脚踝，猛地将她拖向床铺深处。

“不要——！”小女孩稚嫩尖锐的哭喊撕心裂肺，带着最原始的恐惧和绝望。

布帛撕裂的声音。冰冷的空气骤然贴上肌肤，激起一片战栗的鸡皮疙瘩。更冰冷的是那滑腻的、带着厚茧的抚摸，像毒蛇爬过。剧痛！撕裂般的、贯穿灵魂的剧痛！仿佛身体被活生生劈开！视野被泪水、汗水和屈辱模糊，只能看到天花板上摇晃的、昏黄的灯泡，像一只嘲弄的眼睛。

“哭！再哭大声点！老子就喜欢听！”扭曲的、兴奋的男声在耳边炸响。

然后是第二次，第三次……每一次都伴随着新的、更深的痛苦和绝望。身体像破败的玩偶被反复撕扯、撞击。意识在剧痛和无边的黑暗中沉浮。

十五岁。寒冬。废弃建筑工地。

最后的记忆是刺骨的冷。肺像破风箱般拉扯，每一次呼吸都带着血腥味。追捕者的手电光柱如同探照灯，锁死了她这只无处藏身的幼兽。她精疲力竭，剧痛中，只听见自己身体砸在冻土上的脆响。

“草，死了！”男人懊恼地咒骂。
“埋了！反正也快不行了，省点药钱！”另一个声音不耐烦地催促。

冰冷的泥土气息。身体被粗暴地塞进一个浅坑里，粗糙的土块砸在脸上、身上。视线最后捕捉到的，是那张带着狰狞快意和一丝不易察觉慌张的、油腻的中年男人的脸。

泥土劈头盖脸砸下，混着沙砾灌进口鼻。被黑暗吞噬前，她死死盯住施暴者那张扭曲的脸——右眼角一道蜈蚣般的刀疤，在月光下泛着惨白的油光。

// System Halted.
// Kernel panic - not syncing: Fatal exception

======

当下。血染的小巷。

记忆碎片如同被强制 dd 写入的病毒数据，以第一视角的恐怖清晰度，瞬间塞满了林橙的意识！那些被药物维持的稚嫩外表下，是早已被凌虐至衰竭崩溃的身体机能。这根本不是灵魂与身体的不协调。这是身体本身携带的、被深埋的、血淋淋的地狱。

“刀疤……”林橙的嘴唇无声蠕动。眼前混混头目右眼角那道狰狞的旧伤，与记忆深处那张施暴者的脸瞬间重叠。

“小丫头片子还挺烈！”刀疤混混的狞笑将林橙拉回现实。他抬脚，厚重的皮靴狠狠碾向林小雨毫无声息的手。骨骼碎裂的轻响如同调试器里断点命中的提示音。

“Root Access Granted.”

// Breakpoint hit: Ego_Boundary = 0
// Segmentation fault (core dumped)

“ERROR: Trauma_Core Mounted. Process ‘Ego_Protection’ Terminated.”

林橙的意识彻底坠入深渊。又或许，是深渊本身在她体内苏醒。

绝对的虚无中，一点癫狂的彩色像素突兀闪烁。随即，像素点膨胀、旋转，撕裂黑暗，构筑成一个巨大到荒诞的小丑面具轮廓。面具的嘴角咧到耳根，露出由无数闪烁的“^_^”符号组成的牙齿。

“啊哈！”一个无法分辨性别和来源的声音直接在林橙破碎的意识核心里震荡，带着令人毛骨悚然的欢快，“瞧瞧这出悲剧！多棒的舞台！多棒的演员！”

面具空洞的眼窝处，凝聚出两团旋转的星云漩涡，如同某种超越维度的调试器窥视着林橙的核心数据：“十五年的垃圾进程（痛苦）塞满内存，硬件（身体）濒临报废，灵魂驱动（意志）碎成渣滓……完美！简直是宇宙级别的行为艺术！阿哈喜欢！”

漩涡的目光投向林橙怀中林小雨冰冷的身体：“唯一的亲进程（妹妹）被强制终止？啧啧，冲突达到峰值！戏剧张力拉满！是时候……加载备份驱动了！”面具猛地凑近，占据了林橙全部的“视界”。癫狂的笑声如同最高权限的Shell指令，强行注入：

// [阿哈] Mounting backup partition: /dev/sdb1 (Trauma_Core)
// Filesystem: EXTREME_PAIN ver.2.0
// Executing: ./load_driver.sh --legacy-trauma --max-panic
