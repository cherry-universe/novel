26. Rollback
现实中的小巷，时间重新流动。

“啊——！！！”

林橙仰天发出一声非人的、混合着无尽痛苦与暴怒的尖啸。纯白的长发无风狂舞，如同燃烧的白色火焰。左眼“Azure Debugger”爆发出前所未有的冰蓝强光，无数0和1的洪流瀑布般倾泻而下，疯狂分析着眼前这绝对无法接受的现实。右眼“Golden Bug”的金黄瞳孔则缩成了两点刺目的、仿佛能点燃虚空的金星。

“Target Locked: Attackers.”
“Target Locked: Environment (Hostile).”
“Target State: Lethal. Elimination Protocol: Absolute.”

混混们脸上的狞笑还未来得及转化为惊愕。

林橙甚至没有抬手。她只是站在那里，怀中抱着林小雨正在冰冷的身体，意识深处，一个带着毁灭一切意志的指令，如同开天辟地的第一行代码，被冰冷地、决绝地敲下：

sudo rm -rf --no-preserve-root /path/to/attackers_and_this_corrupted_world/*

“Command Executed: Level MAX (SS). Resource: Soul_Burning.”

嗡——！

无法形容的、绝对的寂静瞬间笼罩了整条小巷。时间仿佛凝固了一帧。

下一刻，以林橙为圆心，一个无形的、透明的力场如同投入水面的石子般急速扩散开来。力场所过之处，物质的存在概念被彻底抹除。

砖墙无声地化为最细小的、连显微镜都无法观测的基本粒子，如同被最高倍的橡皮擦擦过，没有烟尘，没有爆炸，只有彻底的消失！地上的垃圾、丢弃的易拉罐、半块砖头、甚至空气……一切非林橙和林小雨存在的物质，都在那透明的涟漪扫过的瞬间，归于彻底的、永恒的虚无。

那几个混混，脸上的表情永远定格在惊恐欲绝的瞬间。他们的身体、衣物、手中的凶器，如同被投入沸水的雪人，从边缘开始，无声无息地分解、消散，连一丝灰烬都没有留下。他们的尖叫被扼杀在喉咙深处，连同存在本身，被彻底从这个世界上“删除”。

不到一秒。

原本狭窄肮脏、堆满杂物的小巷，连同两旁的墙壁、头顶的夜空，彻底消失不见。取而代之的，是一个直径超过十米、边缘光滑如镜、深达数米的半球形巨坑！坑壁呈现出一种非自然的、仿佛被最高精度激光切割过的熔融石英质感，光滑得能映出天空扭曲的倒影。

绝对的寂静。绝对的虚无。

唯有巨坑的中心，林橙跪在那里，怀中抱着林小雨冰冷的身体。她雪白的长发垂落，沾满了妹妹后脑渗出的、已然开始凝固的暗红血迹，如同雪地红梅，刺目惊心。巨坑边缘光滑的曲面倒映着她小小的、颤抖的身影，像一个被遗弃在宇宙废墟中的玩偶。

“小雨……”林橙的声音嘶哑得如同砂纸摩擦。巨大的能量爆发几乎抽空了她的一切，连灵魂都仿佛在燃烧后只剩下灰烬。前世被凌虐致死的冰冷绝望，与今生眼睁睁看着至亲在怀中流逝的无边痛苦，如同两条冰冷的毒蛇，死死缠绕着她的心脏，越收越紧。

左眼的冰蓝光芒黯淡下去，分析停止。因为分析的结果只有一个：死亡。冰冷的、不可逆转的死亡。右眼的金色光芒却如同回光返照般再次亮起，带着一种近乎疯狂的、不顾一切的执念。

“不……不能接受……”她低下头，额头抵着林小雨冰凉苍白的额头，“哥哥…姐姐…还没……还没带你去看新的番剧……还没给你买……合身的内衣……” 断断续续的话语，是灵魂碎片在哀鸣。

突然，一个念头，如同黑暗中唯一燃烧的火种，骤然点亮了她即将熄灭的意识核心！

“Rollback.”

如同溺水者抓住最后一根稻草。如同系统崩溃前最后的、不顾一切的热重启。

“回滚！把时间……回滚！” 林橙猛地抬起头，右眼的金光炽烈到仿佛要融化自身。她无视了身体濒临崩溃的警报，无视了灵魂深处传来的、即将彻底瓦解的剧痛。她将自己残存的、因灵魂燃烧而变得无比“高维”和纯粹的生命本源能量，如同开闸泄洪般，疯狂地、不顾一切地灌注进右眼“Golden Bug”的核心。

目标只有一个：林小雨！
状态：生！
时间锚点：10秒之前！袭击发生的前一刻！

“给我——回去啊！！！”

意念如同咆哮的代码洪流，带着最决绝的意志，冲向了那冰冷残酷的现实壁垒：

btrfs subvolume snapshot /reality/@system /reality/@system_backup_before_disaster // 为现实创建快照（理论锚点）
btrfs subvolume delete /reality/@system // 删除当前灾难状态
btrfs subvolume snapshot /reality/@system_backup_before_disaster /reality/@system // 将快照回滚为当前状态
--target /reality/entities/Lin_Xiaoyu // 目标锁定：林小雨
--timestamp_offset -10s // 时间偏移：负10秒
--force // 强制执行，无视一切规则与代价！
--resource ALL // 支付资源：一切！

“WARNING: Executing Temporal Rollback Operation (Partial Reality Rewrite). Resource Cost: CRITICAL (Life_Force/Ego_Boundary). Proceed? (y/N)”

“Y! Y! Y!!!” 林橙的灵魂在无声地呐喊，每一个“Y”都如同灵魂被剜去一块！

指令生效！

嗡——！

一道比之前湮灭更加柔和、却更加深邃的、带着奇异时间波纹的白色光晕，从林橙右眼流淌而出，瞬间包裹住了她怀中的林小雨。

光晕之中，时间仿佛倒流的胶片。

林小雨后脑那狰狞的伤口，如同被一只无形的手温柔地抚平。翻卷的皮肉、碎裂的骨骼、破裂的血管，以肉眼可见的速度逆向复原、弥合！流出的、沾染在林橙衣服上的鲜血，如同拥有生命般倒流回伤口之内！她苍白如纸的脸色迅速恢复红润，冰冷的体温如同被重新注入暖流，迅速回升！

林小雨长长的睫毛微微颤动了一下，胸口开始出现微弱的起伏。

成功了！

然而，代价是毁灭性的。

林橙右眼的金光如同风中残烛，剧烈地闪烁了几下，彻底熄灭。她体内那因暴走而沸腾的能量核心，如同被瞬间抽干了所有燃料的引擎，发出一声无声的哀鸣，彻底沉寂下去。所有光芒——异能的辉光、生命的火光、意识的灵光——从她小小的身体里瞬间抽离。

她的身体失去了所有支撑，如同被剪断了提线的木偶，软软地、无声地向后倒去，重重地摔在冰冷光滑的坑底地面上。纯白的长发铺散开来，与坑底冰冷的镜面融为一体。她的眼睛空洞地睁着，左眼的冰蓝和右眼的金黄都失去了所有神采，只剩下死寂的灰暗。

能量水平：0% (CRITICAL)
核心温度：过低 (Hibernation/Shutdown Initiated)
意识状态：深度昏迷 (Kernel Panic: Unrecoverable Error)
灵魂：Ego_Boundary Fragmentation Detected.

她倒在了刚刚恢复呼吸、生命体征正迅速稳定的妹妹身边。就像一个耗尽了最后一丝电量，被强制关机的精密仪器。世界陷入一片死寂的黑暗。
