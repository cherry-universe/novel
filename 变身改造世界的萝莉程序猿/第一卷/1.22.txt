22. B+
九月下旬的风，已然带上了刀锋般的凉意，卷起人行道上枯黄的落叶，打着旋儿撞在异能管理局“人才公寓”冰冷的合金窗框上，发出沙沙的轻响。

训练依旧在继续，但“熔炉”里的气氛明显不同了。郑峰出现在训练场监控室的时间越来越少，即使出现，眉宇间也锁着一道深刻的沟壑，眼神比平时更加锐利，像淬了火的刀锋，透过单向玻璃扫视林橙时，带着一种前所未有的审视和…凝重。训练指令依旧严苛，但那些不断加压的“微调”停止了，取而代之的是一种维持现状的、近乎僵硬的模式。林橙甚至能感觉到，那些施加在她身上的、无形的监控“探针”，其扫描频率和强度都提升了一个等级。

一种无形的、沉重的低气压，如同即将降临的暴风雨云层，沉沉地压在管理局的上空，也压在林橙的心头。她体内那点可怜的能量，在这种压抑的氛围下，恢复得更加缓慢。

这天下午的训练结束得异常早。林橙几乎是爬出“熔炉”的，能量堪堪维持在18%的危险线上。她扶着冰冷的墙壁，艰难地走向更衣室，每一步都像踩在棉花上。就在她推开更衣室门的瞬间，一个高大挺拔的身影如同铁塔般堵在了门口。

是郑峰。他似乎刚从某个紧急会议中脱身，制式风衣的下摆还带着室外的寒气，身上散发着一种硝烟与金属混合的、生人勿近的气息。他脸色沉凝，眼神如同鹰隼，牢牢锁定在林橙身上，那目光仿佛带着实质的重量，让林橙本就虚弱的身体又往下沉了沉。

“林橙。”他的声音低沉，带着不容置疑的命令口吻，比训练场里的指令更加冰冷，“接下来一段时间，停止一切非必要的异能使用。禁止离开公寓区域。最大程度降低自身能量场活跃度。明白？”

林橙心头一凛。这种前所未有的严厉警告，像一道冰冷的电流窜过脊椎。她勉强抬起头，冰蓝色的左眼在疲惫中艰难聚焦：“…原因？”她的声音嘶哑。

郑峰没有直接回答，只是微微眯起眼睛，那锐利的目光仿佛要穿透她的颅骨，直视她灵魂深处那SS级的“核心”：“‘清道夫’出现了。B+级，初步判定有向A级演化的趋势。坐标：城西废弃工业区边缘。特征…高度吻合理论模型。”他顿了顿，每个字都像冰冷的铁块砸在地上，“目标明确，行动轨迹…具有强烈的指向性。”

指向性？指向什么？答案呼之欲出。

S级吸引源！

林橙的脸色瞬间褪尽了最后一点血色，苍白得像她自己的头发。一种冰冷的恐惧感攫住了她，比训练后的虚脱更加深入骨髓。是她…是她这个所谓的“SS级”引来的！就像那头摧毁了她们家的铁手！那个夜晚，家园崩塌的轰鸣、妹妹惊恐的眼神、养父母照片被掩埋在废墟下的绝望…如同被强行唤醒的噩梦程序，瞬间涌入脑海！Memory Access: Trauma_Event_2029.8.20 - [ACTIVATED]。身体不受控制地微微颤抖起来。

“我…我没有…”她下意识地想辩解，想说自己一直努力压制着能量场，想说郑峰的训练已经把她榨干得像个空壳。但在郑峰那双洞悉一切、毫无情感波动的眼睛注视下，任何辩解都显得苍白无力。

“没有？”郑峰的声音陡然拔高了一丝，带着金属摩擦般的冷硬，“‘铁手’为什么精准地锁定了你的家？这次出现的‘清道夫’，它的能量场共鸣图谱，和你在上次蚀泥怪事件中残留的‘凝固场’特征，相似度超过78%！这难道是巧合？”他上前一步，巨大的压迫感如同实质的重力场，让林橙几乎喘不过气。“你，林橙，就是这座城市目前最高能的‘灯塔’！无论你愿不愿意，承不承认！”

他俯视着眼前这个因为恐惧和虚弱而显得更加娇小的白发少女，语气斩钉截铁：“特勤三队即刻出发清剿。在我回来之前，你，给我待在公寓里，保持‘静默’状态（Low Power Mode）。这是命令，也是为了你和你妹妹的安全。别让我分心处理更大的麻烦。” 说完，他不再看林橙，猛地转身，风衣下摆划出一道凌厉的弧线，大步流星地消失在走廊尽头，留下一股冰冷的、混合着铁锈和硝烟的气息。

更衣室里只剩下林橙一个人，靠着冰冷的金属储物柜，身体微微发抖。郑峰的话语像冰冷的铁锤，一遍遍敲打着她试图封闭的认知——她就是灾难的源头！是吸引那些可怕怪物的磁石！妹妹小雨…小雨还每天走在那条越来越冷清、越来越黑暗的放学路上！

[ALERT] Direct Threat Correlation Confirmed: USER == S-Class Beacon!
[ERROR] Emotional Subsystem Overload! Fear Response: MAX!
[WARNING] Energy Core Instability Detected! Fluctuation Amplitude Increasing!
视野里，冰蓝色的Azure Debugger模式不受控制地疯狂刷屏，各种警告信息如同病毒般爆发。

不行！不能这样！必须知道外面发生了什么！必须确认那怪物的动向！不是为了郑峰，是为了小雨！

一股强烈的冲动驱使着她。林橙猛地站直身体，无视身体的抗议和郑峰的命令，左眼冰蓝光芒暴涨，右眼金芒也强行点亮！她集中起几乎枯竭的精神力，用意念输入了一条从未尝试过的、范围极大的感知指令：
dmesg --follow --level=err,crit,alert,emerg // 实时监控城市级“内核”紧急日志
journalctl -u global_energy_field.service -f -p 3 // 追踪城市能量场服务，优先级3（错误）及以上
tcpdump -i eth0 -nn -v 'port 53' // 尝试抓取能量场的“DNS”解析信息？纯粹瞎猜！

指令发出的瞬间，如同将一根烧红的探针狠狠捅进了她本已脆弱不堪的能量核心！

“呃啊——！”林橙闷哼一声，眼前骤然一黑，剧痛如同电流般贯穿全身！她双腿一软，跪倒在地，双手死死抠住冰冷的金属柜门才没让自己彻底倒下。Energy: 18% -> 10%! CRITICAL!

视野在剧烈的疼痛和能量枯竭的眩晕中疯狂闪烁、扭曲。冰蓝色的数据流不再是清晰的信息，而是化作一片混沌的、充满尖锐噪音的漩涡！无数破碎、混乱、扭曲的片段如同海啸般涌入她的意识：

—— 刺耳的、非人的金属摩擦咆哮，仿佛来自地底深处！[LOG] CRIT: Abnormal Acoustic Signature Detected (Industrial Zone West)! Frequency: Subsonic/Supersonic Mix!

—— 一闪而过的、令人极度不安的冰冷意志，如同冰冷的扫描光束扫过城市边缘！[LOG] ALERT: Unidentified High-Level Cognitive Pattern Detected! Intent: [PURGE]?
—— 混乱的、带着血腥味的画面碎片：扭曲的钢筋，燃烧的车辆残骸，某种巨大、沉重、布满尖锐棱角的金属肢体在烟尘中若隐若现… [ERROR] Corrupted Memory Fragment Injected! Source Unknown!

信息量太大！太混乱！太危险！如同试图用一台老旧的386电脑实时解析一场核爆的数据流！

“噗！”林橙喉咙一甜，一口鲜血喷在了冰冷的金属柜门上，绽开刺目的猩红。[FATAL ERROR] Buffer Overflow! Core Dump Imminent!
强制终止！
她凭借最后一丝清醒，用尽全部意志力发出了最原始、最暴力的中断指令：
kill -9 -1 // 终止所有进程！强制退出！

嗡——！

脑海中的信息洪流如同被强行掐断的电源，瞬间陷入一片死寂的黑暗。剧烈的反噬如同重锤砸在灵魂上！林橙眼前彻底一黑，身体软软地向前倒去，额头重重磕在储物柜冰冷的金属棱角上，发出一声闷响。鲜血顺着她苍白的额角缓缓流下，混合着嘴角的血迹，滴落在冰冷的地面。

她蜷缩在更衣室冰冷的地板上，意识在剧痛和虚脱的边缘沉浮。视野里只剩下最后一条模糊的、断断续续的红色警告：
[KERNEL PANIC] Segmentation Fault! Core Dumped…
Energy: 5%… System… Unresponsive…
冰冷的恐惧感，如同毒藤，缠绕着她仅存的意识，比身体的伤痛更加刺骨。清道夫…它来了…为了她而来…而小雨…小雨还在外面…
