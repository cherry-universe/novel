4. 异能登记处

第二天上午，达利安市异能管理局，异能登记处。

听完两人费劲巴拉的解释，苏清禾一屁股坐回工位，猛灌了一大口水，眨巴着那双小眼睛，满脸写着难以置信。

干了十年登记工作，什么稀奇古怪的异能她没见识过？可眼前这情况，真真是破天荒头一遭。

《异能导论》写得明明白白：异能五大类——元素操控、肉体强化/变异、精神念力、空间/时间、辅助/功能。而且通常觉醒都在20岁之前，过了这村基本就没这店了。

可眼前这个白毛小萝莉，横看竖看也不像她自称的25岁啊！更离谱的是，她的异能根本塞不进那五大类的任何一个框框里。硬要归类的话，大概……勉强能沾点“肉体强化/变异系”的边？

虽说觉醒变异系异能导致外貌大变的例子也有，可谁能想到……连性别都变了啊！

苏清禾心里疯狂吐槽，但作为资深打工人，脸上愣是绷住了，只呆滞了两秒，脑子就熟练地开启多线程模式：一边录入信息，一边刷着伪装成360广告的短视频。

“好嘞，最后一步，”她敲完键盘，努努嘴示意，“把手放测试水晶球上，测测异能等级。”

“丸辣！”林橙心里咯噔一下。代码操控现实……这能力太惊世骇俗了，万一水晶球显出什么异状，自己怕不是要被抓去当小白鼠切片研究？

不过她的担心很快变成了多余。

目光触及水晶球，一行代码悄然浮现，旁边还有个闪烁的光标：

color=BLACK;

林橙意念微动，将光标移到“BLACK”上。紧接着，一行提示缓缓出现：

enum COLOR { BLACK, RED, ORANGE, YELLOW, GREEN, BLUE, PURPLE, PINK };

她瞬间了然。（背景设定：异能等级EDCBAS对应水晶球颜色橙黄绿蓝紫粉，红色代表无异能，黑色则是待机状态。）

就在指尖触碰到冰凉水晶球表面的刹那，林橙意念疾闪，迅速将 color 的值改成了 YELLOW，随即感觉体内能量被抽走了一丝。

嗡--

水晶球瞬间爆发出明亮而稳定的黄色光芒。

“搞定，这是你的新证件。”苏清禾利落地完成登记，指了指旁边的取卡口。

一旁的林小雨迫不及待地把卡抽了出来，“快让我看看！”

【姓名：林橙】【性别：女】【民族：夏】【异能：永久美化自身外貌】【异能等级：D】【证件号：21020220**********】

苏清禾本着职业操守，打算在异能栏如实填写“性转”，但架不住林橙和林小雨的强烈抗议，加上考虑到当事人未来的社会融入问题，最终妥协成了这个含蓄的版本。

“哇！豪堪！”

林小雨抱着证件左摸右看，卡姿兰大眼睛死死盯着证件照，口水都快流出来了。那模样看得林橙一愣一愣的，仿佛第一次见识到妹妹这种“痴汉”形态。

======

回家的路上。

夕阳将两人的影子拉得老长。林橙别扭地扯了扯身上明显小一号的连衣裙裙摆——这是林小雨“慷慨解囊”的“珍藏品”，据说是她初中时代的战袍。

布料紧绷，裙摆短得堪忧，林橙感觉自己像被硬塞进了一个精致却尺寸不对的礼盒里，每一步都走得小心翼翼，生怕动作幅度稍大就听见布料撕裂的悲鸣。

“哥…呃，姐！感觉怎么样？有没有哪里不舒服？能量够不够？要不要喝点功能饮料补充一下？”林小雨像只兴奋过度的小麻雀，围着她蹦蹦跳跳，目光黏在她那头柔顺的白发和新面孔上，几乎没移开过。

“还行…就是这裙子…”林橙无奈叹气，“勒得慌，而且总觉得下面…凉飕飕的。”

“习惯就好啦！你看你现在多漂亮！比那些顶流明星都耀眼！”

林小雨麻利地掏出手机，“来来来，笑一个！记录这历史性的一刻！我的‘新’姐姐！”

林橙下意识地想躲，终究拗不过妹妹的过分热情，只好僵硬地扯出一个比哭还难看的笑容。

“咔嚓！”

清脆的快门声响起。林小雨满意地审视屏幕：夕阳的金辉勾勒出林橙精致的侧脸轮廓，银白长发仿佛被镀上了一层暖融融的光晕，只是那笑容……僵硬得如同刚学会控制面部肌肉的机器人。

“哇！绝美！姐，你这颜值原地出道，绝对秒杀全场！”林小雨捧着手机，眼睛亮得像发现了稀世珍宝。

林橙刚想吐槽妹妹这八百倍厚的滤镜，脑海中却毫无征兆地闪现出一个网址：

xxxxx://xxxxxx.xxx/xxxxxxxxxxx/xxxxxxxxxxx

几乎是条件反射，林橙的意念光标瞬间锁定了那行地址。念头微动，如同在终端输入命令，她下意识地在网址前敲入了 “fire狐”和空格，然后按下了意念中的“回车”键。

“嗡——”

林小雨手里的手机屏幕猛地一亮，原本的相机界面瞬间消失，一个顶上缺了一角的红色圆环包裹蓝色圆球的图标一闪即逝——火狐浏览器自动启动，页面飞速加载。

“咦？！”林小雨吓了一跳，差点把手机扔出去。低头一看，屏幕上赫然是一个名为“Dress”的 GitHub 项目仓库。她随手点进一个目录，一张二次元风格的小裙子图片弹了出来。

“什么鬼？手机中毒了？”林小雨狐疑地戳戳屏幕，试图关闭浏览器，页面却纹丝不动。她猛地抬头看向林橙。

林橙自己也懵了。她只是脑子里“想”了一下……这能力也太“智能”了吧？连执行环境（手机上的火狐）和操作对象（妹妹的手机）都自动关联锁定了？她赶紧在心里疯狂默念：“exit! close! 关闭！快给我关掉！”

念头刚落，林小雨手里的手机屏幕瞬间黑屏，紧接着又自动亮起，稳稳地回到了锁屏界面，仿佛刚才那几秒的失控从未发生。

“……”林小雨看看恢复如初的手机，又看看一脸无辜（实则心虚）的林橙，眼睛瞪得溜圆，嘴巴慢慢张成一个完美的“O”型，随即爆发出惊天动地的兴奋：“哇靠！！姐！你的异能还能隔空操控手机？！这也太酷炫了吧！这是什么神仙能力！是不是还能帮我抢限量版手办、破解游戏金手指？？”

林橙被她这天马行空的脑洞弄得哭笑不得，连忙摆手：“停停停！打住！别瞎想！刚才…纯属意外！大概…是异能不稳定？对，刚觉醒，不稳定！”她赶紧搬出这个万能借口。

“哦——”林小雨拖长了调子，眼神里闪烁着“我懂，我都懂，你不用解释”的狡黠光芒，显然没信，但很识趣地没再追问，只是宝贝似的把手机揣回兜里，小声嘀咕着，“意外好啊……这种意外请务必多来点……”

林橙扶额叹息，感觉妹妹的接受能力和脑补能力简直比自己这新得的异能还要离谱。

夕阳的暖意渐渐消散，晚风捎来一丝凉意。林橙忍不住又扯了一下那勒得她快喘不过气的裙腰，感觉像被一条过分热情的蟒蛇死死缠住了腰身。短得令人心慌的裙摆，让她每一次抬腿都如同在走钢丝，凉风毫无阻碍地灌入，激起一层细密的鸡皮疙瘩。

“嘶…小雨，这裙子…真的没有尺码正常点的‘珍藏品’了吗？”林橙的声音里带着点绝望的哀求。

“哎呀，忍忍嘛，多好看呀！你看路人的回头率多高！”林小雨指着旁边一个骑着自行车、频频回头差点一头撞上电线杆的小伙子，得意洋洋，“这就证明我的眼光超绝！再说了，姐，”她挥舞着林橙那张新鲜出炉的身份证件，像举着胜利的旗帜，“你现在可是官方认证的‘永久美化自身外貌’D级异能者！外加我猜至少是超S级的‘代码具现’大佬！”

“……”林橙看着证件上那行“永久美化自身外貌”的字样，再低头看看身上这件勒得窒息、布料薄透、裙摆反人类的初中生“珍藏品”，只觉眼前一黑，前途无亮。

她低头望着地上被拉长的两个影子。妹妹的影子活泼地蹦跳着，充满了生机。而自己的影子……纤细、陌生，套在紧窄不合身的裙装轮廓里，别扭地挪着小步。一阵风突兀地卷过，影子上的裙摆瞬间被高高掀起，林橙吓得一个激灵，慌忙并拢双腿，动作僵硬得如同刚上好发条的木偶。
