20. "熔炉"
“情书风波”过去三天了，林橙依旧处于一种强烈的社会性死亡后遗症中。公寓那扇薄薄的门板，成了隔绝外界羞耻宇宙的叹息之墙。她把自己裹在宽大的灰色连帽卫衣里，像一只试图缩回壳里的受惊寄居蟹，连去客厅倒水都尽量贴着墙根走，生怕在走廊那头撞见隔壁那位技术宅邻居——余胜军。

那天打印机疯狂吐出的几十份措辞肉麻、署着她名字的情书，如同梦魇的雪花片，覆盖了整个办公室。郑峰那张万年冰封、刚毅如磐石的脸上，嘴角抽搐的幅度简直能申请吉尼斯世界纪录。他最终“仁慈”地没收了所有“罪证”，并给了林橙几天“免打扰假期”。但这“假期”在林橙看来，更像是公开处刑后的流放。

“这跟被 rm -rf / --no-preserve-root 了有什么区别？”林橙蜷缩在电脑椅上，把脸深深埋进膝盖，发出闷闷的、带着电音质感的哀嚎。屏幕上，一个巨大的、鲜红的“error”提示框正冷酷地占据着整个主显示器。她的Arch Linux系统，在一次常规的 sudo pacman -Syu 滚动更新后，毫无意外地……滚挂了。

滚挂的原因？一个内核模块与某个新驱动版本冲突，导致Xorg服务直接罢工。屏幕一片漆黑，只剩下终端里不断刷新的错误日志，像是在嘲笑她此刻狼狈的人生。

“内核恐慌！Kernel Panic！我的人生也是Panic！”林橙烦躁地抓挠着自己那头纯白色的炸毛长发，左眼冰蓝色的Azure Debugger模式自动开启，视野里瞬间被刷屏的、带着详细内存地址和调用栈的报错信息淹没，如同黑客帝国里的绿色瀑布流。“…drm…nouveau…symbol not found…undefined reference…这他喵的到底是谁的锅？”

她尝试了几次journalctl -xb查看启动日志，又用Live USB引导进去试图修复依赖冲突，但烦躁的情绪如同不断叠加的进程锁，让她根本无法冷静分析。失败的挫败感混合着情书事件的余烬，在胸腔里闷烧。

“姐？”林小雨的脑袋从门缝里探进来，元气满满的脸上带着一丝小心翼翼，“还在跟你的‘战舰’搏斗呢？开学第一天感觉怎么样？”她今天穿着崭新的校服，黑发短而精神，眼神亮晶晶的，充满了高三新开端的兴奋感。

林橙有气无力地抬了下头，右眼金黄色的Golden Bug模式都懒得开：“感觉？感觉像是系统核心被注入了一个不可逆的、名为‘高三’的恶性病毒，即将开始长达一年的高负载运行，并伴有间歇性情绪崩溃风险。”她指了指屏幕，“看，我的‘战舰’先崩溃了。”

林小雨噗嗤一笑，挤进房间，好奇地看着满屏的代码报错：“哇哦，这就是传说中的‘滚挂’？看起来比我们数学卷子的最后一道大题还复杂。”

“复杂？这叫优雅的混乱！”林橙试图找回一点资深程序员的尊严，但很快又泄了气，“算了，跟你解释不清。反正就是，我的开发环境炸了，所有项目停摆，包括那个该死的、郑峰要求的‘异能稳定性报告’初稿。”她一想到那些文书表格，胃里就一阵翻江倒海。

林小雨理解地点点头，随即又兴奋起来：“姐，你知道吗？我们班新来了个转学生，超帅的！而且好像还是体育特长生！”她顿了顿，小脸微红，凑近林橙耳边压低声音，“还有还有，隔壁班的那个女生，今天又偷偷给我塞了手工饼干…百合花形状的哦！”

林橙面无表情地听着妹妹分享校园“情报”，右眼不自觉地扫描了一下小雨书包侧袋里那个包装精致的小盒子，视野里立刻弹出一个框：

fastfetch
Object: Handmade Cookies
Shape: Lily (百合属)
Potential Intent: [ROMANTIC_INTEREST] Probability: 85.7%
Sender: Classmate Wang (隔壁班王同学)
Warning: Contains high levels of sugar and potential emotional attachment.

她嘴角抽搐了一下：“林小雨同学，友情提醒：高三的核心KPI是GPA（Grade Point Average），不是CP（Coupling Pair）。请合理分配系统资源，避免进程优先级错乱导致核心转储（Core Dump）。”

“知道啦知道啦，技术宅姐姐！”林小雨做了个鬼脸，随即又有点担忧地看着林橙，“姐，你这几天…还好吧？余大哥他…好像也挺尴尬的，今天早上在电梯口碰到他，他头都快低到胸口了，招呼都没打就溜了。”

提到余胜军，林橙感觉脸上刚退下去的热度又“噌”地一下回来了。她烦躁地挥挥手，像在驱赶一个顽固的bug：“别提他！Segmentation fault级别的错误！永久性内存泄漏！我现在只想chmod 000（权限全禁）他的门！”

“唉，可怜的余大哥，明明只是撞见了你调试内衣…”林小雨小声嘀咕，看到林橙杀人的眼神立刻举手投降，“好好好，我不说了！那你今天还出门吗？郑队长不是说下午要去训练场吗？”

林橙看了一眼电脑右下角的时间，又感受了一下体内那如同老式硬盘般缓慢运转的能量核心——cat /proc/energyinfo 的意念指令反馈回一个孤零零的 Energy: 45%。郑峰安排的所谓“能力适应性训练”，本质上就是高强度、高耗能的“实战模拟”。美其名曰提升控制力，实则就是变相的极限压榨和监控。每次训练完，她都感觉自己像被dd命令全盘擦写了一遍，只剩下空荡荡的物理躯壳。

“去，能不去吗？”林橙认命地叹了口气，从椅子上滑下来，“那位郑大队长，他的眼神就像gdb的断点，早就钉死在我这个‘潜在高危进程’上了。不去？我怕他直接kill -9（强制终止）。”

她套上那件万年不变的灰色卫衣，兜帽拉得很低，几乎遮住上半张脸，只露出线条紧绷的下巴和几缕不安分的白毛。走出房门时，她刻意放轻了脚步，耳朵却像雷达一样捕捉着隔壁的动静。很好，一片寂静。邻居进程状态：[SLEEPING] or [NOT_HOME]。她松了口气，拉着林小雨迅速溜出了公寓楼。

异能管理局的地下训练场，代号“熔炉”。冰冷、空旷、巨大，空气里弥漫着淡淡的臭氧和金属摩擦的味道。四周是高强度合金墙壁，顶部的照明灯模拟着不同强度的自然光。场地中央，几个形态狰狞、由某种活性合金构成的“靶机”正发出低沉的嗡鸣，它们的关节处闪烁着代表不同威胁等级的红光。

郑峰早已等在那里。他依旧穿着那身笔挺的制式服装，身姿挺拔如标枪。看到林橙这副恨不得把自己缩进地缝里的样子，他刚毅的脸上没有任何表情，只是抬手指了指场地中央。

“今天的目标：在持续干扰环境下，维持对三个移动靶机的‘冻结’指令超过五分钟。同时，准确识别并清除随机出现的‘友方单位’干扰信号。”他的声音冷硬得像敲击键盘的回车键，毫无波澜。“干扰等级：D-。开始准备。”

林橙深吸一口气，压下心头的烦躁和身体深处传来的疲惫感。她走到指定的启动位置，摘下兜帽，纯白色的炸毛在训练场惨白的光线下显得有些刺眼。她闭上眼，集中精神。

左眼冰蓝光芒亮起，Azure Debugger模式启动。视野瞬间切换，整个世界被解析为流动的数据洪流。移动的靶机不再是实体，而是一个个由复杂参数构成的动态模型，它们的运动轨迹被预测为一条条带着概率值的虚线。场地四周，无形的干扰场如同背景噪音，被标注为[ENVIRONMENT_NOISE_LEVEL: D-]。右眼金芒流转，Golden Bug模式就绪，意念中的“命令行”光标闪烁着，等待输入。

“训练开始。”郑峰的声音如同启动指令。

嗡——！

三个靶机瞬间启动，以刁钻的折线高速冲向林橙！同时，一股强烈的、带着混乱无序特征的能量波动从四面八方涌来，如同无数细针扎向她的精神感知。视野中的数据流瞬间变得模糊、抖动，预测轨迹线疯狂闪烁、断裂！

[WARNING] Sensory Input Degradation! Noise Suppression Required!
[ERROR] Target Trajectory Prediction Confidence < 45%!

“啧！”林橙咬牙，强行稳住心神。意念急速输入：
#pragma GCC optimize("O3", "unroll-loops") // 思维加速优化
set_speed(target_1, 0);
set_speed(target_2, 0);
set_speed(target_3, 0);

三道无形的指令如同锁链，精准地缠绕上三个高速移动的靶机。它们的动作骤然僵直，关节处发出刺耳的金属摩擦声，仿佛被冻结在时空琥珀之中。巨大的能量消耗感瞬间袭来，林橙感觉身体一沉，视野边缘开始发黑。Energy: 38% -> 32%。

干扰场并未停止，反而如同潮水般一波波加强，试图冲刷掉她施加的“冻结”力场。靶机在僵直中剧烈震颤，表面的红光疯狂闪烁，随时可能挣脱束缚。

[WARNING] External Interference Amplifying! Control Field Stability: 72%... 68%...
[ALERT] Target_1 Breakout Probability: Rising! 15%... 27%...

就在这时，一个微弱的、带着特定识别码的信号源突然出现在林橙的感知边缘。它伪装成一个无害的漂浮物，但Azure Debugger立刻弹出了高亮警告：

// Object: Simulated Neutral Drone
// Signal Pattern: [FRIENDLY_ID_ALPHA] (Spoofed)
// Actual Threat Level: [D] (Decoy/Distraction)
// Recommendation: Ignore or Mark Safe.

是郑峰说的“友方单位干扰”！林橙的神经紧绷到了极限。维持三个靶机的冻结已经让她如同超频的CPU般滚烫，这突如其来的干扰信号如同一个恶意插入的线程，试图抢夺宝贵的处理资源。她必须快速判断！

右眼金光一闪，意念指令几乎在识别完成的瞬间发出：
mark_safe(friendly_decoy_alpha); // 标记为安全，忽略

指令生效，那个虚假的“友方信号”在感知中被标注为绿色无害，干扰的噪音略微降低了一线。她得以将更多精力集中在对抗外部干扰和维持冻结场上。

时间一分一秒流逝。汗水浸湿了林橙额前的白发，顺着苍白的脸颊滑落。她单薄的身体在巨大的能量和精神负荷下微微颤抖，像一张拉满到极限的弓。训练服的后背早已湿透。Energy: 22%... 18%... 红色的低能量警告如同催命符般在视野中闪烁。

“还有十秒。”郑峰冰冷的声音如同倒计时。

林橙咬紧牙关，榨取着身体最后一丝能量。视野中的数据流开始出现雪花噪点，靶机的震颤幅度越来越大。Control Field Stability: 41%... Target_1 Breakout Probability: 89%!

五…四…三…二…一！

“时间到。训练结束。”

嗡鸣声和干扰场瞬间消失。三个靶机如同断电的玩具，僵在原地，关节处的红光缓缓熄灭。施加在它们身上的“冻结”力场也随之解除。

林橙双腿一软，再也支撑不住，膝盖重重磕在冰冷坚硬的合金地面上。她双手撑着地，大口大口地喘着粗气，眼前阵阵发黑，汗水滴落在地面，洇开一小片深色。Energy: 15%。身体像被掏空，每一个细胞都在发出过载的悲鸣。

郑峰走到她面前，居高临下地看着。他的眼神锐利如鹰，扫过林橙狼狈的样子，又看向旁边仪器记录下的庞大数据流——能量消耗曲线、控制稳定性波动、干扰耐受阈值、反应时间…最终，他的目光停留在林橙强行标记“友方干扰”时那近乎本能的、毫秒级的准确判断上。

“控制力，D。稳定性，D-。能耗比，极差。”他毫不留情地给出评价，声音平板无波，像是在宣读一份不合格的产品检测报告。“不过…对信息噪音的过滤和关键威胁的识别判断，尚可。”他顿了一下，似乎在斟酌用词，“像一台…能耗恐怖、随时可能宕机，但核心算法勉强能用的古董级服务器。”

林橙连翻白眼的力气都没有了，只能在心里疯狂刷屏：

// System Message: F*ck You!
// Error: Energy Critical. Sarcasm Module Unavailable.
// Recommendation: Find nearest horizontal surface and hibernate.

郑峰似乎没兴趣接收她的“系统消息”，转身离开，只留下一句：“明天同一时间。训练强度提升10%。把你的‘系统日志’整理好，我要看。” 他的背影消失在通往监控室的气密门后。

林橙趴在地上，脸贴着冰凉的地板，感受着那点微不足道的凉意试图给过热的“CPU”降温。重启中…请稍候… 她只想就地把自己 shutdown -h now（立即关机休眠）。

拖着仿佛灌满了铅的身体回到公寓，林橙连澡都不想洗，只想一头栽倒在床上。然而，刚走到自己房门口，一个身影有些局促地挡在了走廊中间。

是余胜军。

他个子很高，此刻却微微佝偻着背，眼神躲闪，手里拿着一个巴掌大小、看起来像某种定制电路板的东西，上面还接着几根数据线。他的脸涨得通红，说话也结结巴巴：

“林…林橙…那个…下午好…不不，晚上好…我…我听说你系统…Arch滚挂了？”他不敢看林橙的眼睛，目光在她卫衣的拉链上飘忽，“我…我以前也经常滚挂…那个…nouveau驱动的问题…我…我这里有个小工具…自己焊的…”

他像是终于鼓起勇气，把那个小电路板往前递了递，语速飞快：“就是个…便携式的内核panic日志抓取和分析器…能绕过Xorg直接读显存帧缓冲…或许…或许能帮你定位到具体的冲突模块…不用谢！放你门口了！”

说完，他像是怕被拒绝或者被林橙那“格式化金属杯”的能力给抹掉，飞快地把那个还带着余温的小工具放在林橙房门口的地毯上，然后逃也似的冲回自己房间，“砰”地一声关上了门。走廊里只剩下他刚才站立处一点淡淡的松香焊锡膏的味道。

林橙僵在原地，兜帽下的脸也是一片滚烫。羞耻感、尴尬，还有一丝…被技术同好精准戳中痛点的微妙感，在她心里搅成一锅粥。她看着地上那个造型粗犷但透着实用主义气息的小工具，最终，还是弯腰捡了起来。

入手微沉，电路板边缘打磨得很光滑，显然是花了心思的。她默默输入 lsusb 意念指令，扫描了一下这个小设备：

// Device: Custom Kernel Panic Sniffer
// Vendor: NEIGHBOR_YU
// Capabilities: FrameBuffer Dump, Kernel Log Parsing, Basic Symbol Resolution
// Power Source: Micro USB / CR2032 Backup
// Status: [READY]
// Estimated Usefulness: High (for current crisis)

“…// Comment: Well... technically useful.” 林橙低声嘟囔了一句，攥紧了那个小工具，推开了自己的房门。房间里，三块显示器依旧漆黑一片，无声地诉说着滚挂的悲剧。她叹了口气，把余胜军的“神器”插上电脑的USB接口。

屏幕上，一串串新的、更清晰的错误信息开始滚动。或许…这个arch还能抢救一下？她拉开椅子坐下，指尖重新放到了键盘上。窗外的夜色渐浓，城市的灯火在远处流淌，公寓里只剩下键盘敲击的清脆声响，以及偶尔传来隔壁林小雨看番剧时压抑不住的惊呼和笑声。新的一天，新的社死，新的bug，新的训练…但生活，这台庞大而混沌的系统，依旧在磕磕绊绊地运行着。
