6. 兼容性

“嗤啦——！”

布料撕裂的脆响，在怪物被收容后的短暂寂静里，显得格外刺耳，比蚀泥怪的嘶吼还让林橙魂飞魄散。腰间猛地一凉，仿佛警报系统在她脊椎里拉响了最高级别的红色警报——走光了！她甚至能感觉到路人的目光像扫描仪一样聚焦在她腰侧那道新鲜的裂口上，以及……那印着半颗草莓的浅色布料边缘。

“呜啊——！姐！你的……你的裙子！”林小雨的尖叫简直能震碎玻璃，与其说是惊恐，不如说是混合了震惊、荒谬和一丝……诡异的兴奋？

林橙的脸瞬间从虚脱的惨白变成了煮熟的虾红。“跑！”她从牙缝里挤出一个字，声音抖得像在编译一个全是内存泄漏的垃圾代码。她一手死死捂住腰间的“门户大开”，另一只手铁钳般抓住林小雨的胳膊，力道之大，妹妹的校服袖子瞬间起了褶子。

这一刻，林橙爆发出了堪比 git push --force 般不顾一切的决绝。她脑子里只有一个目标：马路对面那家灯火通明的“霓裳物语”。

林小雨瞬间领悟，反手架住姐姐（感觉像在拖一个随时会散架的人形服务器），爆发出体育中考百米冲刺的潜力。林橙感觉自己像个被强行拖拽的破旧人偶，每一次迈步，腰间的裂口都在“嘶嘶”地嘲笑她，凉风嗖嗖地往里灌，提醒她这具身体和灵魂的兼容性有多糟糕。

左眼视野边缘又开始不受控地刷屏：

[WARNING] Physical integrity compromised: Lower_Torso_Fabric_Integrity. Threat level: EMBARRASSMENT_CRITICAL.
[ERROR] Kernel panic? Negative. Fabric panic confirmed. System stress: MAXIMUM.

“哐当！”林小雨用肩膀暴力开门，姐妹俩像两颗出膛的炮弹砸进服装店凉爽的空调风里。门口的感应器“叮咚”一声，店员那句“欢迎光临”刚吐出一半就卡在了喉咙里。

眼前的景象堪称行为艺术：一个银白炸毛、蓝金异瞳的绝美少女，小脸煞白（残留着能量耗尽的虚弱），却以一种极其扭曲的姿势死死捂着腰间裂开的裙子，细腿抖得像是在运行一个死循环；旁边一个扎马尾的少女气喘吁吁，脸红得像苹果，眼神里闪烁着“我家服务器炸了但炸得好萌啊”的诡异光芒。

“试衣间！最大的！立刻！马上！”林小雨语速快得像开了2倍速，目光如探照灯般扫过衣架，完全无视店员呆滞的表情。

店员的目光精准地落在林橙捂着的腰侧、那狰狞的裂口以及……若隐若现的草莓图案上。职业假笑瞬间切换成紧急救援模式：“这边请！快！”她小跑着把两人引向最里面宽敞的试衣间。

林橙几乎是把自己“投掷”进试衣间的，“砰”地一声甩上门落锁，后背重重抵在冰凉的墙壁上，大口喘气，仿佛刚从DDoS攻击中幸存下来。安全区！暂时的。

她低头审视这身“战损”装备——妹妹初中时的旧裙子，像一件严重缩水的容器，勒得她生不如死，如今终于彻底罢工。松开手，那道裂口咧着嘴，无声地嘲讽着她“林橙”的新身份。

“姐！开门！急救包来了！”林小雨的声音隔着门板传来，带着一种清空购物车般的雀跃。

门缝里塞进来几件衣服。林小雨压低但兴奋的声音：“快试试！这件粉的！超可爱！这件蓝的！显白！还有这件！带小蝴蝶结和蕾丝的！萌炸了！快穿上让我拍……呃，看看效果！”

林橙拎起那堆轻飘飘、缀满花边、饱和度突破天际的“萌系装备”，眼角和嘴角同时开始不受控地抽搐。这审美……跟她衣柜里那些洗得发白的纯色T恤和工装裤，简直是银河系的两端！她甚至能预见到穿上之后视野里的弹窗：

[ALERT] FATAL STYLE MISMATCH! User preference profile (Comfort++ / Practicality+) conflicts with proposed solution (Kawaii Overload 9000).
[SUGGESTION] Execute sanity_check? (Y/n)

“林！小！雨！”林橙从牙缝里挤出妹妹的名字，努力让声音听起来不像在磨刀，“宽松！纯棉！T恤！运动裤！越像装土豆的麻袋越好！Understand？！”

“懂懂懂！可是姐！”林小雨的声音充满了循循善诱，“你现在是行走的萌点发射器！是SSR级美少女！机会难得！你就试一件嘛，就一件！我保证不拍照……呃，最多一张？”门缝里又塞进来一件——总算是一件看起来像是人类能穿的、印着简单灰色几何线条的白色宽松棉质连衣裙。

林橙长叹一声，认命地开始与布料搏斗。脱掉那件“凶器”旧裙子的瞬间，她舒服得差点呻吟出声。然而，当她试图把新裙子套过头时，这具新身体的“驱动不兼容”问题再次暴露无遗。手臂该抬多高？腰该怎么扭？大脑发出精确指令，肢体却像是在用300ms延迟的远程桌面操作，反应迟钝，动作笨拙。

[NOTICE] Motor control driver reloading... Latency: HIGH (Estimated 350ms).
[WARNING] Legacy muscle memory (Lincheng v2.0) incompatible with current hardware (Lincheng v1.18). Driver rollback required.（注意，这里是伏笔）

她咬牙切齿，终于把那件白色连衣裙套好。柔软透气的棉布包裹着身体，宽松的剪裁终于让她找回了呼吸的自由权。她长长地、满足地舒了一口气，准备推门出去宣告胜利。

就在她转身抬脚的瞬间，鞋尖精准地勾住了地上那件粉色蕾丝裙的飘逸裙摆。

“丸辣！”念头刚闪过，大脑紧急向身体发送了“紧急制动！后退半步！”的指令。然而，在神经信号的传输路上，某个关键节点似乎发生了严重的丢包错误。她的右脚踝以一种极其反关节、反物理的姿态，猛地向内一崴。

“嗷——！”一声短促的萝莉痛呼，尖锐的疼痛从脚踝直冲天灵盖。林橙身体瞬间失衡，像个失控的陀螺一样踉跄着向旁边倒去，“咚”的一声闷响，手肘狠狠磕在了试衣间坚硬的墙壁上。

“姐？！怎么了？摔了？受伤了？”门外的林小雨声音瞬间拔高，充满紧张。

林橙疼得龇牙咧嘴，扶着墙站稳，小心翼翼地活动了一下脚踝。“……没骨折！死不了！”她没好气地回应，这破身体真是bug百出。

她带着一种劫后余生的疲惫感，推开了试衣间的门。

“哇——！”林小雨的眼睛瞬间像探照灯一样亮起，“姐！绝了！清冷破碎感小白花！这裙子配你这头发这眼睛，绝配！比那些花里胡哨的好看一万倍！”

宽松的白色棉裙确实中和了银发异瞳的些许异质感，显得干净、清爽，甚至透出一点（林橙本人毫无自觉的）脆弱美感。

林橙无视了妹妹夸张的赞美，只感到一阵解脱：“就它了。结账，立刻。”

林小雨蹦蹦跳跳去付钱，还不忘回头叮嘱：“姐，你脚没事吧？慢点走哦！”她付完钱，像献宝一样把装着新裙子的袋子塞给林橙，然后变魔术似的从自己小背包里掏出两个印着卡通创可贴，“喏，给！虽然没破皮，但磕那一下肯定青了，贴上这个，草莓味的！可可爱爱！”

林橙看着那粉嫩嫩、印着小小草莓图案的创可贴，再看看自己身上同样简约的白裙，又想起腰侧曾经暴露过的草莓内裤边缘……她嘴角抽了抽，最终还是默默接过，撕开一个，笨拙地贴在了隐隐作痛的手肘上。
