21. 日常
时间在“熔炉”训练场日复一日的极限压榨中，被拉伸、扭曲。日历无声地翻到九月中旬，窗外的梧桐叶边缘开始染上些许焦黄。

郑峰对林橙的训练，如同一个冷酷无情的压力测试程序，每一次都将参数向上微调一点点。干扰等级从D-稳步提升到C#；需要同时冻结的靶机数量增加到四个，甚至偶尔会加入第五个作为“隐藏惊喜”；“友方干扰”信号出现的频率更高，伪装也更加精巧；训练时长被严格卡死在五分钟，但其中的“有效对抗时间”占比要求越来越高。

每一次训练，对林橙而言都是一场灵魂层面的马拉松。走出“熔炉”时，她往往脸色惨白如纸，脚步虚浮得如同踩在云端，汗水浸透的训练服紧紧贴在身上，勾勒出少女单薄却因持续透支而微微颤抖的轮廓。cat /proc/energyinfo 反馈的数字越来越触目惊心：18%，15%，有一次甚至跌破了10%的警戒线，眼前发黑，几乎是被林小雨和训练场的医护人员架出来的。恢复时间也从最初的几小时，延长到了大半天，甚至一整天都处于一种低功耗的“待机”状态，反应迟钝，食欲不振，只想蜷缩在黑暗里。

郑峰对此的回应，永远只是监控屏幕上那些跳动的数据和冰冷的评价：
“能量输出峰值不稳定，波动幅度超过安全阈值15%。”
“对复合干扰源的屏蔽效率低于预期，存在明显感知漏洞。”
“精神韧性不足，低能量状态下判断力下降显著。”
“核心问题：能耗比依然无法接受。SS级？目前表现仅达到D+级门槛。”

林橙已经懒得在心里刷屏反驳了。她的“系统消息”模块仿佛也因长期低能量而宕机，只剩下一种近乎麻木的疲惫。她感觉自己像一块被反复充放电、内阻越来越大的旧电池，每一次训练都在加速损耗着某种不可逆的东西。

身体的异样感也在这日复一日的极限压榨中悄然滋生。有时，在训练场高强度输出后，她会感到一阵突如其来的、毫无缘由的冰冷恐惧，如同坠入深海，瞬间淹没意识。视野里甚至会短暂地闪过一些破碎、扭曲的影像——布满灰尘的服务器机柜缝隙？键盘上凝固的暗红色液体？这些碎片一闪即逝，快得无法捕捉，只留下剧烈的心悸和一身冷汗。还有一次，路过管理局食堂后厨，看到厨师用巨大的金属钩子悬挂起半扇猪肉，那冰冷的金属光泽和猩红的肉质，让她胃里一阵翻江倒海，强烈的眩晕和呕吐感瞬间袭来，几乎是逃离了现场。

[WARNING] Unexplained Physiological Stress Response Detected!
[ERROR] Memory Access Violation! Corrupted Data Fragment?
[HYPOTHESIS] Residual Traumatic Memory from Previous System Crash (Death Event)?
[ACTION] Suppress. Classify as System Noise. Focus on Core Functionality (Survival/Training).

她把这一切归咎于重生时那场诡异的“系统重装”残留的bug，以及训练带来的巨大身心压力。如同程序员对待某些难以复现的诡异bug，选择了暂时搁置和忽略，用一层厚厚的“实用主义”代码将其封装隔离。

======

唯一能穿透这层厚重疲惫与自我隔离的，是林小雨。

高三的课业压力如同无形的巨石，沉沉地压在了林小雨稚嫩的肩膀上。她回家越来越晚，眼下时常带着淡淡的青黑，曾经活力四射的短发似乎也少了几分跳跃的劲头。但每次推开家门，看到蜷缩在沙发上、像只耗尽电量的大型玩偶般的姐姐时，她脸上总会立刻绽开大大的笑容，如同阴霾里骤然投射的阳光。

“姐！我回来啦！”她踢掉鞋子，书包随意一甩，就扑到林橙身边，献宝似的从口袋里掏出东西——有时是一小块学校小卖部热销的巧克力，包装纸被捂得有点融化；有时是一朵路边捡到的、形状奇特的落叶；有时只是几句学校里听来的、并不好笑的冷笑话。

“看！今天物理随堂测，我居然及格了！老班看我的眼神都像见了鬼！”林小雨盘腿坐在地毯上，一边啃着苹果，一边眉飞色舞地讲述，试图驱散姐姐眉宇间的沉郁。“还有还有，隔壁班那个王同学，今天体育课跑八百米，最后冲刺的时候摔了个大马趴，假发都飞出去老远！噗哈哈哈…原来他是为了遮住地中海才戴假发的！”

林橙裹着毯子，靠在沙发里，右眼Golden Bug模式都懒得开，只是用左眼冰蓝色的Azure Debugger模式安静地扫描着妹妹。视野里，林小雨的生命体征数据平稳，但精神压力指数（根据微表情、语速、瞳孔细微变化等分析）在安全阈值的高位徘徊。Stress_Level: 73% (High School Senior Mode)。她听着妹妹叽叽喳喳的声音，感受着身边传来的、属于活人的温暖体温和淡淡的洗发水味道。这声音和温度，如同稳定运行的背景进程，一点点抚平她因训练而紊乱的“内核”。

“姐，你知道吗？”林小雨的声音忽然低了一点，带着一丝不易察觉的抱怨，“最近放学路上，感觉好冷清啊。以前那条小吃街可热闹了，现在好多店都关得早，路上人也少了好多。特别是从学校后门抄近路穿过那个小公园…路灯坏了好几盏，黑黢黢的，一个人走还真有点瘆得慌。”

[ALERT] Environmental Factor: Reduced Lighting, Reduced Human Traffic.
[ASSESSMENT] Potential Security Risk: Elevated.
[ACTION] Recommend: Avoid Low Visibility Routes. Increase Vigilance.

林橙闭着的眼皮微微动了一下。一股细微的电流般的警觉感顺着脊椎爬升，但很快被沉重的疲惫压了下去。她伸出手，有些笨拙地揉了揉妹妹的头发，指尖传来柔软蓬松的触感。

“怕…就…走大路。”她的声音因为虚弱而显得格外轻，带着点气音，像接触不良的喇叭。“或者…等我…恢复…接你。”这句话说出来，连她自己都觉得没什么底气。郑峰严格控制着她的外出，更别提在低能量状态下去人多眼杂的地方。

“算啦算啦！”林小雨立刻又扬起笑脸，反过来抓住林橙的手，“你好好休息！我可是要保护姐姐的人！再说了，我可是带了‘秘密武器’的！”她神秘兮兮地从书包里掏出一个粉红色的、带小兔子挂件的警报器，用力按了一下，刺耳的蜂鸣声瞬间响彻客厅。

林橙被这突如其来的噪音刺激得一个激灵，左眼视野里瞬间刷过一片警告：
[WARNING] High Decibel Noise Detected! Sensory Overload!
[ERROR] Auditory Processing Unit Overdriven!
“林！小！雨！”她咬牙切齿，想伸手去掐妹妹的痒痒肉，却因为脱力动作慢得像树懒。

林小雨大笑着跳开，关掉警报器，脸上是恶作剧得逞的狡黠：“看！多响亮！方圆五十米的坏人都得吓跑！姐你安心当你的睡美人吧，骑士小姐会安全到家的！”她蹦蹦跳跳地去厨房倒水，哼着不成调的动漫主题曲。

林橙看着妹妹活泼的背影，冰蓝色的左眼中，那高企的压力指数似乎悄然回落了那么一点点。Stress_Level: 68%。身体的疲惫依旧沉重如山，但胸腔里某个冰冷坚硬的部分，似乎被这吵嚷的、充满生气的温暖，悄然融化了一角。// System Message: Core Temperature Rising. Emotional Subsystem… Functional.。

与余胜军的关系，也在一种心照不宣的、技术宅式的默契中，缓慢地解冻。

那次“内核分析器”成功帮助林橙定位并修复了滚挂的系统（虽然过程依旧痛苦），像一把钥匙，打开了两人之间那扇因“情书”和“内衣调试”而冰封的门。余胜军似乎彻底放下了尴尬，或者说，将那份尴尬转化成了纯粹的技术交流欲。

他开始“不经意”地在林橙门口放一些小东西。有时是一个U盘，里面是他收集整理的、关于异能能量场波谱分析的最新开源论文（虽然林橙怀疑管理局内部有更机密的资料）；有时是一张打印出来的、思路清奇的故障排查流程图，针对林橙某次训练后抱怨的“能量流间歇性阻塞”问题；最夸张的一次，是一个用乐高积木拼成的、极其简陋的、代表“蚀泥怪”的小模型，旁边贴着一张便签纸，上面用歪歪扭扭的字写着：“模拟凝固指令？试试改变局部熵增速率？纯理论瞎猜…[笑脸]”。

林橙一开始还绷着脸，假装没看见。但技术宅的本能最终战胜了羞耻心。某天训练回来，累得灵魂出窍，看到门口又放着一个贴着便签的小型传感器模组。

// Device: Miniature Energy Flux Monitor (Prototype v0.2)
// Function: Passive Monitoring of Personal Energy Core Fluctuations (Non-invasive)
// Power: Micro USB. Output: CSV over Serial.
// Note: Calibration might be off. Use with skepticism. - YU

林橙盯着那个小东西看了几秒，最终还是弯腰捡了起来。// Object Acquired. Scanning…。回到房间，她鬼使神差地把它插在了电脑上。简陋的串口终端里，开始滚动一行行实时数据，正是她体内那如同风中残烛般摇曳的能量核心读数。虽然精度粗糙，波动剧烈，但确实提供了一个客观的、可视化的“血条”参考，比她自己意念 cat /proc/energyinfo 的主观感受要直观一些。

第二天，林橙出门时，在余胜军的门口放了一罐她囤积的、提神效果堪比液氮的“程序员特浓咖啡”。罐子上贴着一张从废代码本上撕下来的纸条，上面画着一个极其潦草的、竖起大拇指的像素风笑脸，旁边还有一行小字：
// Coffee: [FUEL_TYPE: HIGH_OCTANE]
// Feedback: Monitor v0.2 - Accuracy: [D-], Usefulness: [B+]. Keep Iterating. - L.C.

没有对话。没有眼神交流。只有冰冷的合金门板和门口交换的小物件。但一种基于共同语言（技术）和共同困境（被郑峰压榨）的、微弱的同盟感，在这无声的“Pull Request”与“Code Review”中悄然建立。当林橙拖着疲惫的身躯回来，看到门口又出现新的“小玩意儿”，或是余胜军收到那罐咖啡后门内隐约传来的一声压抑的欢呼时，那沉重的脚步，似乎也轻快了那么微不足道的一丝。

林橙靠在冰冷的公寓走廊墙壁上，感受着体内缓慢回升到25%的能量。窗外，城市的灯火在渐浓的秋意中显得温暖而遥远。隔壁房间里，林小雨翻动书页和偶尔的嘟囔声隐约传来。楼下，余胜军似乎在焊接着什么新的小东西，松香的味道顺着门缝飘出。

训练的痛苦、低能量的虚弱、身体莫名的异样、对未来的不安…如同后台堆积的错误日志。但此刻，这点滴汇聚的、来自妹妹和邻居的微弱暖流，如同系统日志里偶尔闪现的 [INFO] Connection Established 提示，让她这台濒临过载的“服务器”，得以在临界阈值之上，维持着最低限度的、带着回响的运转。// System Status: CRITICAL, but… OPERATIONAL.
